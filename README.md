# Stone Paper Scissors Game

A simple and interactive **Stone Paper Scissors** game built using **HTML, CSS, and JavaScript**. Play against the computer and try to be the first player to reach **5 points**.

---

## 🚀 Features

* 🎲 Random computer choice
* 🧑 Player vs Computer gameplay
* 📊 Live score tracking
* 🏆 First to 5 points wins
* 🔄 Restart game functionality
* 💬 Dynamic result messages
* 🎨 Clean and responsive user interface

---

## 🛠️ Technologies Used

## 🎮 How to Play

1. Click **Stone**, **Paper**, or **Scissors**.
2. The computer randomly selects its choice.
3. The winner of the round earns one point.
4. The first player to reach **5 points** wins the game.
5. Click **Restart Game** to start a new match.

---

## 🏆 Game Rules

* 🪨 Stone beats Scissors
* 📄 Paper beats Stone
* ✂️ Scissors beats Paper
* Same choice = Draw
